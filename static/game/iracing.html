推奨動作環境

iRacingを快適にプレイするための推奨スペックです。60fps以上の安定したフレームレートを実現できます。

項目 推奨スペック
GPU（グラフィックボード） Gaming graphics card with at least 8GB of DEDICATED memory (some examples but not limited to Nvidia 2080 TI, AMD Radeon VII)
CPU（プロセッサ） 8 core, more CPU (some examples but not limited to: Intel Core 9-10980XE @ 3.0GHz, Intel Core i9-9900KS @ 4.00GHz, AMD Ryzen 9 3900X,
RAM（メモリ） 32GB
ストレージ 100GB（SSD推奨）
OS Windows 10/11 (64bit)

最低動作環境

起動可能な最低限のスペックです。低設定・30fps前後での動作となるため、快適性は推奨スペック以上を推奨します。

項目 最低スペック
GPU A gaming graphics card with at least 4GB of DEDICATED memory – Some examples (but not limited to): Nvidia GeForce GTX 1060, AMD Radeon 570
CPU 4 core CPU, better – Some examples (but not limited to): AMD FX-6300, Intel Core i5-4430, Intel Core i5-2320, AMD Ryzen 3 1200
RAM 16GB

よくある質問

RTX 3060でiRacingは動きますか？

RTX 3060は推奨GPU（Gaming graphics card with at least 8GB of DEDICATED memory (some examples but not limited to Nvidia 2080 TI, AMD Radeon VII)）と同等以上の性能です。1080p/中〜高設定で60fps以上の快適なプレイが可能です。

予算8万円でiRacingの推奨スペックを満たせますか？

GPU（RTX 4060: 約5.5万円）+ CPU（Ryzen 5 7600: 約3万円）で推奨スペックを満たせます。RAMは別途1万円程度必要です。

iRacingの最低動作環境は？

最低GPU: A gaming graphics card with at least 4GB of DEDICATED memory – Some examples (but not limited to): Nvidia GeForce GTX 1060, AMD Radeon 570、最低RAM: 16GB。ただし、低設定・30fps前後での動作となります。快適なプレイには推奨スペック以上を推奨します。
