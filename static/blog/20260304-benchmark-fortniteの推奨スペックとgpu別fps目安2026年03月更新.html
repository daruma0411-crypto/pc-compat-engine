Fortnite 推奨スペック・GPU別FPS目安【2026年3月最新版】

最終更新：2026年03月04日
新年度・新生活に向けたPC購入シーズン到来！GDC 2026開催月ということもあり、Fortniteにも新たなアップデートが期待されます。本記事では、Fortnite 推奨スペックとGPU別FPS目安を、最新の性能スコア（価格.com調べ・2026年03月04日時点）をもとに詳しく解説します。

Fortniteの動作環境（公式要件）

※Fortniteの公式スペック情報はEpic Games公式サイトに掲載されています。以下は一般的に知られている要件をまとめたものです（Steam配信タイトルではないため、Steam公式スペックは存在しません）。

最低スペック

  • OS：Windows 10 64-bit
  • CPU：Intel Core i3-3225 / AMD equivalent
  • メモリ：8GB RAM
  • GPU：Intel HD 4000 / NVIDIA GeForce GT 640 / AMD Radeon HD 7850相当
  • VRAM：2GB
  • ストレージ：空き容量 約50GB以上

推奨スペック

  • OS：Windows 10/11 64-bit
  • CPU：Intel Core i5-7300U / AMD Ryzen 3 3300U以上
  • メモリ：16GB RAM
  • GPU：NVIDIA GeForce GTX 1060 / AMD Radeon RX 580以上
  • VRAM：6GB以上
  • ストレージ：SSD推奨・空き容量 約50GB以上

Fortniteは比較的軽量なタイトルですが、Unreal Engine 5（Nanite・Lumen）対応以降は高画質設定時の負荷が大幅に増加しています。競技設定（低〜中設定）であればミドルクラスGPUでも快適ですが、最高画質＋高解像度ではハイエンドGPUが必要です。
