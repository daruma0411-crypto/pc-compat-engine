「Fortnite」が動かない時の対処法7選

2026年03月04日 | PC互換チェッカー

Fortniteが動かない・起動しない時の対処法7選【2026年最新版】

新年度・新生活シーズンで新しいPCを購入したのに「Fortniteが動かない」「Fortniteが起動しない」というトラブルに悩んでいませんか？ここでは、Fortniteのトラブルを解消するための具体的な対処法を7つ、初心者の方にもわかりやすく解説します。

1. スペック不足の確認

Fortniteが動かない原因として最も多いのがPCスペックの不足です。まずはお使いのPCが以下の要件を満たしているか確認しましょう。

Fortniteの一般的な最低動作環境（目安）：

  1. OS：Windows 10 64bit 以降
  2. CPU：Intel Core i3-3225 / AMD equivalent 以上
  3. メモリ：8GB RAM 以上
  4. GPU：Intel HD 4000 / NVIDIA GeForce GT 630 / AMD Radeon HD 6670 以上（VRAM 2GB以上推奨）
  5. ストレージ：空き容量 40GB 以上（SSD推奨）

確認手順は簡単です。キーボードの「Windowsキー + R」を押し、「dxdiag」と入力してEnterを押してください。「DirectX診断ツール」が開き、CPU・メモリ・GPUの情報を確認できます。スペックが不足している場合は、パーツの増設や買い替えを検討しましょう。

2. グラフィックドライバの更新

Fortniteが起動しない・画面が真っ暗になるといったトラブルの多くは、グラフィックドライバの古さが原因です。

  1. NVIDIAの場合：NVIDIA公式サイトから「GeForce Experience」をインストールし、最新ドライバに更新
  2. AMDの場合：AMD公式サイトから「AMD Software」をダウンロードして更新
  3. Intelの場合：Intel公式サイトから「ドライバー＆サポート・アシスタント」を利用

更新後は必ずPCを再起動してからFortniteを起動してください。

3. DirectX / Visual C++ 再配布パッケージの再インストール

Fortniteの動作にはDirectXおよびVisual C++ランタイムが必須です。これらが破損していると起動しないことがあります。

  1. Microsoft公式サイトから「DirectX エンドユーザー ランタイム Web インストーラ」をダウンロードして実行
  2. 同じくMicrosoft公式サイトから「Visual C++ 再配布可能パッケージ」の最新版（x64およびx86）をダウンロードしてインストール
  3. インストール完了後、PCを再起動

4. ウイルス対策ソフトの例外設定

ウイルス対策ソフトがFortniteの実行ファイルをブロックしてしまうケースがあります。

  1. お使いのウイルス対策ソフトの設定画面を開く
  2. 「例外」「除外」「ホワイトリスト」などの項目を探す
  3. Fortniteのインストールフォルダ（通常は「C:\Program Files\Epic Games\Fortnite」）を例外として追加
  4. Windows Defenderを使用している場合は「Windowsセキュリティ」→「ウイルスと脅威の防止」→「設定の管理」→「除外の追加」から設定

5. 管理者権限で実行

権限不足でFortniteが起動しないこともあります。以下の手順を試してください。

  1. Fortniteの実行ファイル（FortniteClient-Win64-Shipping.exe）を右クリック
  2. 「プロパティ」→「互換性」タブを開く
  3. 「管理者としてこのプログラムを実行する」にチェックを入れる
  4. 「適用」→「OK」をクリックして再度起動

6. ファイルの整合性チェック

ゲームファイルが破損していると正常に動作しません。Epic Games Launcherで修復しましょう。

  1. Epic Games Launcherを開く
  2. 「ライブラリ」からFortniteを見つける
  3. Fortniteの右下にある「…（三点メニュー）」をクリック
  4. 「確認」を選択する
  5. 検証が完了するまで待つ（数分〜数十分かかる場合があります）

破損ファイルが見つかった場合は自動的に再ダウンロード・修復されます。

7. セーブデータ（設定ファイル）の移動・リセット

設定ファイルの破損が原因でFortniteが起動しないこともあります。設定ファイルをリセットしてみましょう。

  1. 「Windowsキー + R」を押し、「%localappdata%」と入力してEnter
  2. 「FortniteGame」フォルダを見つけて開く
  3. 「Saved」フォルダ内の「Config」フォルダを探す
  4. 「Config」フォルダをデスクトップなど別の場所にコピー（バックアップ）
  5. 元の「Config」フォルダを削除してからFortniteを起動

起動後、グラフィック設定やキー設定はデフォルトに戻りますが、ゲームの進行データ（スキンやバトルパスなど）はサーバーに保存されているため消えることはありません。

それでも解決しない場合は

上記7つの対処法をすべて試してもFortniteが動かない・起動しない場合は、お使いのPC環境固有の問題が考
