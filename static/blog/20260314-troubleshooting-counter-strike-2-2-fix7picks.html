カウンターストライク2が起動しない！焦る前に試してほしい7つの対処法

「カウンターストライク2が動かない」「カウンターストライク2が起動しない」──このページにたどり着いたあなたは、今まさにそんなトラブルに直面しているのではないでしょうか。筆者も以前、Steamで「プレイ」を押した瞬間にクラッシュして真っ暗な画面とにらめっこした経験があります。あの焦りは本当によくわかります。でも大丈夫です。カウンターストライク2のトラブルは、意外とシンプルな手順で解決できるケースがほとんどです。ここでは初心者の方でもすぐ試せるように、7つの対処法を順番にご紹介します。

対処法1：スペック不足の確認

まず最初に確認してほしいのが、お使いのPCがカウンターストライク2の最低動作環境を満たしているかどうかです。一般的に公開されている要件としては、OSがWindows 10以降、CPUがIntel Core i5-6600K / AMD Ryzen 5 1600相当以上、メモリが8GB以上、GPUがNVIDIA GeForce GTX 1050 Ti / AMD Radeon RX 560（VRAM 2GB以上）相当が目安とされています。「設定」→「システム」→「バージョン情報」でCPUとメモリを、「デバイスマネージャー」→「ディスプレイアダプター」でGPUを確認してください。もしスペック不足であればパーツの換装を検討しましょう。筆者のおすすめはコスパ最強のRTX 4070ですが、予算を抑えたい方にはGeForce RTX 4060（価格.com調べ・最安値帯で約42,000円前後）も十分な選択肢です。また、メモリ不足ならCrucial DDR4-3200 16GB×2枚セット（価格.com調べ・約8,500円前後）への増設が手軽でおすすめです。ここで原因がわかれば儲けもんです。

対処法2：グラフィックドライバの更新

スペックに問題がなければ、次に疑うべきはグラフィックドライバです。古いドライバのままだとカウンターストライク2が起動しないケースは非常に多いです。NVIDIAならGeForce Experience、AMDならAMD Softwareを開いて、最新ドライバに更新してください。筆者の環境ではドライバを2バージョンほど放置していたら突然起動しなくなり、更新だけであっさり直ったことがあります。更新後は必ずPCを再起動しましょう。これだけで解決したら本当にラッキーです。

対処法3：DirectX / Visual C++ 再配布パッケージの再インストール

ゲームの実行に必要なランタイムが壊れている場合も、起動しない原因になります。以下の手順を試してみてください。

  1. Microsoft公式サイトから「DirectX エンドユーザーランタイム Webインストーラー」をダウンロードして実行します。
  2. 同じくMicrosoft公式サイトから「Visual C++ 再配布パッケージ」の最新版（x64）をダウンロードし、インストールします。
  3. インストール後、PCを再起動してからカウンターストライク2を起動してください。

地味な作業ですが、ここで直るケースも意外と多いです。焦らず一つずつ進めましょう。

対処法4：ウイルス対策ソフトの例外設定

Windows DefenderやサードパーティのセキュリティソフトがCS2の実行ファイルをブロックしている場合があります。「Windowsセキュリティ」→「ウイルスと脅威の防止」→「設定の管理」→「除外の追加」で、CS2のインストールフォルダ（通常は C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive）を例外に追加してください。筆者も過去にセキュリティソフトが原因でゲームがブロックされていたことがあり、例外設定だけで一発解決しました。ここまでくれば半分以上の原因は潰せているはずです。

対処法5：管理者権限で実行

権限不足で起動できないパターンもあります。手順は簡単です。

  1. Steamのライブラリでカウンターストライク2を右クリック →「管理」→「ローカルファイルを閲覧」を選択します。
  2. 表示されたフォルダ内の「cs2.exe」を右クリックし、「プロパティ」を開きます。
  3. 「互換性」タブで「管理者としてこのプログラムを実行する」にチェックを入れ、「OK」を押します。

たったこれだけです。ここで動いたら本当に儲けもんですよ。

対処法6：ゲームファイルの整合性チェック

ダウンロード中のエラーやアップデートの失敗でファイルが破損していることがあります。Steamのライブラリでカウンターストライク2を右クリック →「プロパティ」→「インストール済みファイル」タブ →「ゲームファイルの整合性を確認」をクリックしてください。破損ファイルがあれば自動的に再ダウンロードされます。数分かかることもありますが、気長に待ちましょう。この方法はカウンターストライク2に限らず、あらゆるSteamゲームのトラブルで使える万能技です。

対処法7：セーブデータ・設定ファイルの移動

ここまで試しても起動しない場合は、設定ファイルの破損が疑われます。CS2のユーザー設定は通常「C:\Program Files (x86)\Steam\userdata\（あなたのSteamID）\730」フォルダ内に保存されています。このフォルダをデスクトップなどにバックアップ（コピー）してから、元の場所のフォルダを削除し、ゲームを起動してみてください。設定は初期化されますが、起動さえすれば後から調整できます。最後の手段ですが、これで直ったという報告も多い方法です。
