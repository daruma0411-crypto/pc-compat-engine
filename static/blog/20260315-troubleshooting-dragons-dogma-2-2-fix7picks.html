ドラゴンズドグマ2が起動しない！動かない時に試す7つの対処法【2026年3月最新】

「ドラゴンズドグマ2が起動しない！」――せっかく購入したのに、プレイボタンを押しても画面が真っ暗、あるいはクラッシュして落ちる。その焦りと悔しさ、痛いほどわかります。筆者も発売直後にまさにこの状態にハマりまして、深夜2時にひとりで唸っていた経験があります。Metacritic 88点の名作を目の前にして遊べないのは本当につらいですよね。でも安心してください。これから紹介する7つの対処法を順番に試せば、ほとんどのケースで解決できます。落ち着いて、ひとつずつ確認していきましょう。

対処法1：スペック不足がないか確認する

まず最初にやるべきは、お使いのPCがドラゴンズドグマ2の要求スペックを満たしているかの確認です。Steam公式に掲載されているスペックは以下の通りです。

【最低スペック】

  • GPU：NVIDIA GeForce GTX 1070 / AMD Radeon RX 5500 XT（VRAM 8GB）
  • CPU：Intel Core i5-10600 / AMD Ryzen 5 3600
  • RAM：16GB

【推奨スペック】

  • GPU：NVIDIA GeForce RTX 2080 / AMD Radeon RX 6700
  • CPU：Intel Core i7-10700 / AMD Ryzen 5 3600X
  • RAM：16GB

確認方法は簡単です。Windowsの「設定」→「システム」→「バージョン情報」でCPUとRAMを、「dxdiag」コマンドでGPU情報をそれぞれチェックできます。筆者の環境ではRTX 4070を使っているので余裕でしたが、以前サブ機のGTX 1060で試したところ見事に起動すらしませんでした。もしGPUが足りない場合、たとえばNVIDIA GeForce RTX 4070は現在59,800円前後（価格.com調べ）で手に入りますし、もう少し予算を抑えるならAMD Radeon RX 7600が39,980円前後（価格.com調べ）で推奨スペックを十分クリアできます。ここでスペック不足が判明して対処できれば儲けものです。

対処法2：グラフィックドライバを最新に更新する

スペックに問題がなければ、次に疑うべきはグラフィックドライバです。古いドライバのままだと、ドラゴンズドグマ2が正常に起動しないケースが非常に多いです。

  1. NVIDIAの方は「GeForce Experience」、AMDの方は「AMD Software: Adrenalin Edition」を開きます。
  2. 「ドライバー」タブから最新版が配信されていないか確認します。
  3. 更新がある場合は「ダウンロード」→「クリーンインストール」を選択して適用します。

筆者も以前、ドライバ更新をサボっていたせいで別のゲームがクラッシュし続けた経験がありまして、更新した瞬間に嘘のように直りました。ドライバ更新だけで直れば、所要時間5分の大勝利です。

対処法3：DirectX・Visual C++ ランタイムを再インストールする

ドラゴンズドグマ2はDirectX 12を使用しています。また、Visual C++ 再頒布可能パッケージが破損していると起動に失敗することがあります。

  1. Microsoftの公式サイトから「DirectX エンドユーザー ランタイム Web インストーラ」をダウンロードして実行します。
  2. 同じく公式サイトから「Visual C++ 再頒布可能パッケージ」の最新版（x64）をダウンロードし、「修復」を選んでインストールします。
  3. 完了したらPCを再起動してください。

地味な作業ですが、これが原因だったというケースは意外と多いです。ここで解決すれば、お金もかからず万々歳ですね。

対処法4：ウイルス対策ソフトの例外設定を行う

Windows Defenderや市販のウイルス対策ソフトが、ドラゴンズドグマ2の実行ファイルを誤検知してブロックしている場合があります。

  1. 「Windowsセキュリティ」→「ウイルスと脅威の防止」→「設定の管理」を開きます。
  2. 「除外の追加または削除」から、ドラゴンズドグマ2のインストールフォルダ（通常は C:\Program Files (x86)\Steam\steamapps\common\Dragons Dogma 2）を丸ごと例外に追加します。
  3. 市販のセキュリティソフトをお使いの場合も、同様にゲームフォルダを除外リストに追加してください。

セキュリティソフトが原因で起動しないのは初心者の方が特に見落としやすいポイントです。ここで直ったら、もう一安心ですよ。

対処法5：管理者権限で実行する

Windowsの権限設定が原因で、ゲームが必要なファイルにアクセスできず落ちるケースもあります。

  1. ドラゴンズドグマ2の実行ファイル（dd2.exe）を右クリックします。
  2. 「プロパティ」→「互換性」タブを開きます。
  3. 「管理者としてこのプログラムを実行する」にチェックを入れて「適用」→「OK」を押します。

たったこれだけで起動するようになることもあります。1分で終わる作業なので、ダメ元で試してみてください。

対処法6：ゲームファイルの整合性をチェックする

ダウンロード中にファイルが壊れていたり、アップデートで一部が欠損することがあります。Steamには便利な検証機能が用意されています。

  1. Steamライブラリでドラゴンズドグマ2を右クリックし、「プロパティ」を選択します。
  2. 「インストール済みファイル」タブを開きます。
  3. 「ゲームファイルの整合性を確認」をクリックします。
  4. 検証が完了するまで数分待ち、問題があれば自動的に修復されます。

筆者はこの整合性チェックで何度救われたかわかりません。破損ファイルが見つかって修復されれば、ほぼ確実に起動するようになります。ここまで来て直れば、長かった戦いもようやく終了です。

対処法7：セーブデータを一時的に移動する

上記すべてを試しても起動しない場合、セーブデータの破損が原因の可能性があります。

  1. エクスプローラーで C:\Users\（ユーザー名）\AppData\Local\Dragon's Dogma 2 フォルダを開きます。
  2. 中にあるセーブデータファイルをデスクトップなど別の場所にコピー（バックアップ）します。
  3. 元のフォルダからセーブデータを削除し、ゲームを起動してみます。
  4. 起動に成功したら、セーブデータを1つずつ戻して破損ファイルを特定しましょう。

セーブデータが消えるのは怖いと思いますが、必ずバックアップを取ってから作業すれば大丈夫です。最後の砦ですが、これで解決したという報告も少なくありません。
