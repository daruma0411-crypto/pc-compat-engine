電源ユニット選び方ガイド

容量の計算方法から80PLUS認証・モジュラー選びまで。PCの安定動作に直結する電源を正しく選ぼう。

⚡電源容量の選び方

電源容量（W数）は全パーツの消費電力合計 × 1.25〜1.3倍を目安にします。20〜30%の余裕を持つことで電源効率が良い領域（50〜80%負荷）で動作させられます。

構成別おすすめ容量目安

  • RTX 4060 + Core Ultra 5：650W以上
  • RTX 4070 + Core Ultra 7：750W〜850W
  • RTX 4080 + Core Ultra 9：850W〜1000W
  • RTX 4090 + ハイエンドCPU：1000W〜1200W

🏅80PLUS認証：効率ランクを選ぼう

80PLUS認証は電源の変換効率を保証する規格です。上位認証ほど効率が高く、無駄な発熱が少なくなります。

  • 80PLUS Standard：80%効率。最低限。
  • 80PLUS Bronze：82%。廉価帯の定番。
  • 80PLUS Gold：87%。コスパ最良・おすすめ
  • 80PLUS Platinum：90%。長期使用なら電気代メリット大。
  • 80PLUS Titanium：92%。最高効率・フラッグシップ向け。

一般的な自作PCには 80PLUS Gold が最もコスパが良くおすすめです。

🔌モジュラー方式の選択

  • フルモジュラー：全ケーブルが着脱式。不要ケーブルをなくしエアフロー向上・配線しやすい。推奨。
  • セミモジュラー：主電源ケーブルは固定、PCIeなどは着脱式。コスパ◎
  • ノンモジュラー：全固定。廉価だがケーブル管理がしにくい。
