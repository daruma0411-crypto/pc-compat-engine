## 背景・目的

現行の「診断型」チャットボットは、コンバージョンまでのステップが長く離脱リスクが高い。
「やりたいことから逆算で提案する」方向にピボットし、購買意欲が高い状態で Amazon/楽天リンクに誘導する。

## モード1：ゲームモード（新規実装）

### フロー

```
🎮 カード選択
    ↓
🤖「どのゲームをしたいですか？予算も教えてください」
    ↓
👤「モンハンワイルズ、予算15万」
    ↓
🤖 Steam games.jsonl参照 → PC構成提案
   ┌────────────────────────────────┐
   │ モンハンワイルズ 推奨構成       │
   │                                │
   │ GPU  RTX 4060 Ti   ¥55,000〜  │
   │ CPU  Core i5-14400F ¥28,000〜 │
   │ RAM  DDR5 32GB      ¥18,000〜 │
   │                                │
   │ 合計目安: ¥13〜15万            │
   │ [🛒 Amazon] [🛍 楽天]  ← 各パーツ │
   └────────────────────────────────┘
    ↓
🤖「144fps狙いならGPUをRTX 4070に上げると安定します」
   （壁打ちの起点：自動でひとこと追加提案）
    ↓
👤「もう少し安くできる？」← 自由に質問
    ↓
購入へ
```

### バックエンド

- エンドポイント: `POST /api/recommend`
- ゲーム名を Claude Haiku で抽出 → `data/steam/games.jsonl` から検索
- 推奨スペック（GPU/CPU/RAM）を既存 products.jsonl と照合
- 予算内で最適な構成を提案
- 各パーツに Amazon/楽天アフィリエイトリンク付き
- 壁打ち用のひとこと追加提案を必ず付ける

### データ依存

- `data/steam/games.jsonl`（Phase 1: Steamスクレイパーで構築）

## モード2：互換チェックモード（既存改善）

### フロー（現行との差分）

```
🔧 カード選択 or テキスト入力
    ↓
👤「RTX 4090をNZXT H200iに入れたい」
    ↓
🤖 診断 → NG判定（変更なし）
    ↓ ← ボタン不要、自動で表示（改善点①）
   ┌────────────────────────────────┐
   │ 💡 代わりにこれはどうですか？    │
   │                                │
   │ GPU変更案       ケース変更案    │
   │ RTX 4070 Ti     Define 7       │
   │ 318mm ✅        491mm ✅       │
   │ [Amazon][楽天]  [Amazon][楽天] │
   └────────────────────────────────┘
    ↓
🤖「RTX 4070 TiならH200iに収まります。
   性能差は約15%。予算を抑えたいなら
   RTX 4070無印もあります」（改善点②：壁打ち誘導）
```

### 改善点まとめ

| 現行 | 改善後 |
|---|---|
| NG後に「GPUを変える」ボタンをクリック | NG判定と同時に自動表示 |
| 「対応GPUの候補です（6件）」 | 「💡 代わりにこれはどうですか？」（提案口調） |
| 代替品リストのみ表示 | AIがひとこと添えて壁打ちを誘導 |
